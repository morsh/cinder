# Cinder

## Prequestisits

- Install [yarn](https://yarnpkg.com/lang/en/docs/install/)

## Installation

If you don't have yarn installed:

```sh
npm install -g yarn
```

```sh
git clone https://github.com/morsh/cinder
cd cinder/
yarn
```

# Running in Development

```sh
yarn start:dev
```

# Run Tests

```sh
yarn test
```

# Build for Production

```sh
yarn build
```

# Run Production Build

```
yarn start
```
